# DOYOUKNOWJU (두유노우주) - 모의투자 웹 서비스 (Backend)

팀 프로젝트 최종 결과물 포트폴리오 정리용 레포입니다.  
가상 투자(모의투자) + 게임요소(경험치/레벨/도전과제/칭호/랭킹) + 커뮤니티(게시판/채팅) 기능을 결합한 서비스입니다.  
자세한 내용은 `docs/FINAL_PROJECT.pdf`(최종 발표 자료) 참고.

- Team: 김동건 / 민경찬 / **유근영(본인)** / 임유택

---

## 핵심 기능
- 주식 Top10: 거래량 Top10 + 거래대금/상승률/하락률/시가총액 탭 전환, 10초 주기 갱신
- 코스피/코스닥 지수 차트: 1h/day/week/month/year 범위 전환
- 커뮤니티: 게시판/댓글 + 인기글(실시간/주간)
- 검색: 자동완성 + 검색 결과(종목정보 + 가격/등락)

---

## 담당 구현 (유근영)
발표자료 기준 본인 담당 기능입니다.

- **주식 TOP10**
  - 실시간 거래량 Top10 + 탭 전환(거래대금/상승률/하락률/시총), 10초 주기 자동 갱신
  - KIS OpenAPI / Naver 데이터 연동 및 응답 필드 차이 안정 처리

- **코스피/코스닥 지수 차트**
  - 메인에서 지수 차트 조회, 1h/day/week/month/year 범위 전환
  - 응답 포맷 변화 대응(시간/가격 파싱 유연화), 빈 응답/오류 시 기존 차트 유지

- **게시판 기능**
  - 자유게시판/종목토론방 목록·상세·작성/수정·댓글
  - 종목 선택 기반 필터링, 실시간/주간 인기글, 마이페이지(내 글/댓글)
  - 정책: 가입 7일 작성 제한, 권한 처리, 비속어 마스킹(프론트/백 분리 적용)

- **검색 기능(자동완성 + 검색결과)**
  - 종목명/종목코드 자동완성
  - 검색 결과에서 종목 정보 + 가격/등락 정보 조회
  - “종목 검색 + 복수 현재가 조회” 2단계 구조, 공통화(debounce/예외처리)
